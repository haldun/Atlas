# Atlas

A Swift codebase visualizer inspired by [Jonathan Blow's work](https://www.youtube.com/watch?v=BqF2SbY99B8&list=PLQe_cPxDPZcp28phDhy4ZnfLdXsAWS8hH). Open a Swift project folder and see its structure as a treemap. Node size reflects code size, color reflects complexity. Click any node to open it in Xcode.
